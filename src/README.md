# The Catalyst (العامل الحفاز) 🧪

The official educational platform for science students.
**منصة العامل الحفاز - بوابتك لعلم أفضل.**

## About the Project
This platform is designed to provide interactive science resources, educational content, and student management for middle school students in Egypt.
تم تصميم هذه المنصة لتقديم موارد علمية تفاعلية ومحتوى تعليمي لطلاب المرحلة الإعدادية في مصر.

## Features
- Interactive Science Modules.
- Student Tracking & Progress.
- Simplified Educational Workflow.
